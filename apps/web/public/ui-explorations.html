What modern mail clients actually do

A survey of Gmail, Apple Mail, Superhuman, Shortwave, HEY, Notion Mail, Spark, Missive, Proton & Canary (2025–2026), distilled into the features and the unique UI ideas worth stealing. The three tabs above turn this into concrete, better-than-current designs.

ClientOrg modelSignature AIUnique UI idea
Gmail Labels (not folders) + category tabs Primary/Social/Promos Gemini: AI Inbox, thread summaries, Help Me Write, AI search overviews Tabbed inbox + 3 density modes
Apple Mail Folders + Categories Primary/Transactions/Updates/Promotions Apple Intelligence: Priority Messages, list-view summaries, Smart Replies Digest view + sender logos + Liquid Glass
Superhuman Split Inbox Important / VIP / News / Calendar Ask AI (inbox+calendar+web), Auto Summarize, Instant Reply ⌘K palette + 100ms keyboard-first
Shortwave Bundles + scheduled delivery AI assistant panel, Ghostwriter, AI todos (press T), NL search, Tasklet automations Assistant panel beside the thread
HEY Imbox / The Feed / Paper Trail Deliberately none in the client Screener + Reply-Later / Set-Aside stacks
Notion Mail Customizable Views (database-style) AI auto-labels, NL filters, AI write, /schedule Saved Views reshape the inbox
Spark Smart Inbox (People / Notifications / Newsletters) +AI assistant, summaries, smart replies "While you were away" unread cards
Missive Shared team inboxes AI Rules autopilot, summarize/translate (Claude/Gemini/GPT) Inline chat + live-cursor co-drafting
Proton Folders + labels + new auto-categories Scribe — runs on-device (local model) Lock-icon encryption indicators
Canary Unified inbox + Focused inbox Copilot sidebar, AI triage, semantic search SecureSend + always-on Copilot

The feature list (table stakes → modern)

  • ✓Triage: snooze / remind-me, archive, swipe actions, undo send, schedule send, "reply later" queue, set-aside / read-later
  • ✓Organize: categories/labels, bundles (group newsletters/receipts), VIP senders, multiple/split inboxes, saved views
  • ✓AI: thread summary at top, smart-reply chips, AI compose in your voice, natural-language search, AI auto-triage, AI to-dos from email
  • ✓Speed: ⌘K command palette, full keyboard nav, <100ms optimistic actions, density modes
  • ✓Trust: SPF/DKIM/DMARC at a glance, tracker blocking, encryption indicators, read receipts
  • ✓Compose: snippets/templates, /slash commands, scheduling availability inline, signatures, send-as identities
  • +Collab (later): shared inboxes, inline comments, assignments, co-drafting

Unique UI ideas — and where they land

  • → A⌘K palette + Split Inbox + keyboard hints — Superhuman speed, Apple-calm surface
  • → BImbox/Feed/Receipts boxes + bottom stacks + Screener + bundles — HEY & Shortwave triage
  • → CAI assistant panel + saved Views + summary/to-dos/smart replies — Shortwave & Notion Mail
  • allSender avatars, AI summary replacing the snippet, SPF/DKIM/DMARC chips, dark-first with density
Current MailKite UI today: a basic 3-pane (rail · list · reading) with unread dots and auth badges. All three explorations keep its bones but add triage, AI, and speed.
